# pePIneBEL – Basisstruktur

Dieses Repository bildet die Grundstruktur des Projekts **pePIneBEL**.  
Es enthält die Kernseiten, die für Identität, Anzeige und Systemstart notwendig sind.

Alle Dateien in diesem Projekt sind **eigenständig erstellt** und nicht aus anderen Repositories übernommen.

---

## 🎯 Zweck des Projekts

**pePIneBEL** dient als einfache, klare und minimalistische Struktur für:

- Identitätsdarstellung  
- Systemstartseite  
- spätere Erweiterungen (Module, Router, Achsen, UI‑Elemente)  

Es ist bewusst leicht gehalten, damit es als Basis für weitere Systeme genutzt werden kann.

---

## 📁 Dateien im Repository

| Datei | Funktion |
|-------|----------|
| `index.html` | Startseite des Systems |
| `ID.html` | Identitätsseite (System‑ID, Struktur, Merkmale) |
| `README.md` | Dokumentation des Projekts |
| `88.html` | (entfernt) war eine alte Testseite |

---

## 🧬 System‑Identität

Die Identität wird in `ID.html` definiert.  
Sie enthält:

- System‑ID  
- Strukturtyp  
- Basisinformationen  
- visuelle Darstellung im Retro‑Monospace‑Stil  

Diese Datei ist vollständig eigenständig und kann jederzeit erweitert werden.

---

## 🧩 Erweiterbarkeit

Das Projekt kann später erweitert werden um:

- Router‑Module  
- JSON‑Konfigurationen  
- Achsen‑Systeme  
- UI‑Scanner  
- 6D‑Matrix‑Module  
- TriAxiom‑Integration  

Die Struktur ist dafür vorbereitet.

---

## 🔧 Installation / Nutzung

Es ist keine Installation notwendig.  
Die Dateien können direkt im Browser geöffnet werden:

- `index.html` → Start  
- `ID.html` → Identität

## 📌 Status

Das Projekt ist minimal, stabil und bereit für Erweiterungen.
